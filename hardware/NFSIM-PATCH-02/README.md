### NFSIM-PATCH-02 — Wound Healing Monitor

**README.** Adhesive patch worn over wound (under transparent secondary dressing) measuring local O₂, pH, temperature, and impedance (exudate amount). Real-time wound state for diabetic ulcer management, surgical incision monitoring.

**SPECIFICATIONS.** Flex PCB with O₂ electrode + ISFET pH + NTC + 4-electrode impedance + nRF52840. 7-day wear. ~$85.

**THEORY.** Healing wounds have characteristic O₂/pH/temp/impedance signatures. Deviation flags infection or chronic non-healing. Patient stays at home; alerts clinician if pattern indicates trouble.

**ASSEMBLY.** Sterile single-use patch. Electrodes/sensors on tissue-contact side with biocompatible spacer.
