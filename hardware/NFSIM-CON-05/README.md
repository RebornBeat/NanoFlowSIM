### NFSIM-CON-05 — Field Sample Collection Kit

**README.** Standardized sample collection tubes + NFC-tagged barcodes for chain-of-custody. Used in field epidemiology, household sampling for genomic surveillance.

**SPECIFICATIONS.** Vacutainer-style tubes (1.5 mL × 6) + DNA/RNA-stabilizing buffer pre-loaded + NFC tag (NTAG216) + tamper-evident seal + shipping bag. ~$8 per kit.

**THEORY.** Sample tagged with NFC + barcode at collection; chain-of-custody recorded; tube linked to patient via tap-to-phone. Buffer preserves DNA/RNA at ambient temperature for 30+ days.

**ASSEMBLY.** Pre-loaded sterile tubes from contract manufacturer; NFC tag affixed at sealing.
