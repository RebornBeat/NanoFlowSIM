### NFSIM-DEL-03 — Microneedle Drug Patch

**README.** Dissolvable PLGA/PVP microneedle array patch for transdermal vaccine, insulin, or hormone delivery. Painless (microneedles only penetrate stratum corneum).

**SPECIFICATIONS.** 10×10 microneedle array, 600 µm length. PLGA dissolvable (vaccine) or PVP rapid-release. Single-use disposable. ~$15.

**THEORY.** Microneedles bypass skin barrier; dissolvable matrix releases drug. Pain-free vs. traditional injection.

**ASSEMBLY.** PLGA/PVP molded in micro-pattern using PDMS molds (academic or contract foundry). Drug + matrix cast; demolded; sterile packaged.
