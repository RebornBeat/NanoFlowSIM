### NFSIM-ABL-01 — RF Cautery Pen

**README.** Handheld 400 kHz radiofrequency cautery for minor surgical hemostasis, electrosurgery in field/remote settings. Safety-interlocked.

**SPECIFICATIONS.** 400 kHz RF, 50W max; monopolar + bipolar modes; foot pedal or finger trigger; safety interlock (impedance monitoring). ~$125.

**THEORY.** RF current heats tissue at electrode contact → coagulation. Used by surgeons for hemostasis during procedures.

**SAFETY.** RF current limit + impedance monitoring; foot pedal kill switch; ground pad fault detection.

**ASSEMBLY.** RF generator board + isolation transformer + pencil handle + cable. Audit-logged each activation.
