#### SPECIFICATIONS.md

**Base Unit:**
| Parameter | Value |
|---|---|
| MCU | STM32H7 (Cortex-M7, 480 MHz) |
| BLE | nRF52840 (BLE central, up to 8 simultaneous connections) |
| USB host | USB 3.0 host controller + 4 USB-A ports |
| 434 MHz | Si4463 ×4 diversity receivers for capsule data |
| Wi-Fi | ESP32-S3 or WL18MK module |
| Ethernet | 1 GbE via WIZnet W6100 |
| NFC | PN5180 ISO 15693 reader |
| Display | 4" capacitive touch IPS (480×800) |
| Storage | 32 GB eMMC + microSD slot |
| Compute connector (for CON-06) | M.2 2280 with custom NanoFlowSIM pinout (PCIe Gen3 ×2 + USB 3.0 + GPIO over 12V supply) |
| Power | 12V/5A wall adapter or PoE+ |
| Dimensions | 220 × 150 × 60 mm |
| BOM | ~$110 (base unit alone) |

**Edge Compute Module (CON-06):** See dedicated entry below.
