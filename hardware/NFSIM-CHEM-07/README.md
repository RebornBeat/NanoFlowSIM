### NFSIM-CHEM-07 — Cortisol Sensor

**README.** Saliva cortisol via competitive ELISA cartridge. Stress monitoring, adrenal function screening.

**SPECIFICATIONS.** Cartridge with HRP-conjugated cortisol competitor; TMB substrate; colorimetric or amperometric readout. ~$90 reader + $8/cartridge.

**THEORY.** Competitive immunoassay: sample cortisol competes with HRP-cortisol for antibody binding. More sample cortisol → less HRP bound → less signal at readout.

**ASSEMBLY.** Cartridge slot reader; LED+photodiode or LMP91000 readout.
