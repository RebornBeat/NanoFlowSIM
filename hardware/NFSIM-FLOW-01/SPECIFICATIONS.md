#### SPECIFICATIONS.md

|---|---|
| Detection method | Lensless holographic imaging (diffraction pattern → AI reconstruction) |
| Light source | Coherent 470 nm LED + 50 µm pinhole (spatial filter) |
| Sensor | Sony IMX477 12.3 MP CMOS (Raspberry Pi HQ Camera) |
| Pixel pitch | 1.55 µm |
| Flow channel | 30 µm × 100 µm × 10 mm PDMS microfluidic |
| Throughput | 1,000–5,000 cells/s |
| Applications | WBC differential, RBC count, platelet count, basic cell morphology |
| Multiplexing | Single-color natural (diffraction); 1-color fluorescence add-on possible |
| Acoustic-focusing variant | Available as add-on PCB for users needing tighter focusing |
| Connectivity | USB 3.0 + BLE 5.0 |
| Power | USB 3.0 5V + optional battery |
| BOM | ~$380 |
| Compute target | Hub+CON-06 or laptop (reconstruction is compute-intensive) |
