### NFSIM-NEURAL-01 — High-Density ECoG Array

**README.** Research-grade high-density ECoG array — flexible Parylene-C microelectrode array + Intan RHD2132 amplifiers ×8 for 256 channels. NOT for human implantation in MVP; veterinary + research model use.

**SPECIFICATIONS.** 256-channel (or up to 1024 with stacking). Parylene-C flex array, 200 µm electrode spacing. Intan RHD2132 (32ch each) ×8. 30 kSPS, 16-bit. Open Ephys compatible. ~$650.

**THEORY.** ECoG = electrocorticography. Subdural electrodes for high-resolution cortical recording. Research-grade preamplification.

**ASSEMBLY.** Custom Parylene-C array fabrication (academic or contract foundry). Intan RHD chips on PCB headstage. Open Ephys interface board for data acquisition.
