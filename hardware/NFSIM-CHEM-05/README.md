### NFSIM-CHEM-05 — Lactate Meter

**README.** Single-analyte handheld lactate meter (sports performance, sepsis screening). Strip-based amperometric.

**SPECIFICATIONS.** Lactate oxidase strip; range 0.5–25 mmol/L; 13s readout; LMP91000-based.

**BOM.** ~$45 (+ strips $2 each).

**THEORY.** Lactate + O₂ + LOx → pyruvate + H₂O₂. H₂O₂ → current at WE.

**ASSEMBLY.** Smaller form factor than CHEM-01; single strip channel.
