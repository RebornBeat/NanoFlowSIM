### NFSIM-CHEM-04 — Lipid Panel

**README.** Cartridge-based total cholesterol, HDL, LDL (calculated), triglycerides from fingerstick.

**SPECIFICATIONS.** Same chassis as CHEM-01. Cartridge has cholesterol oxidase + lipase + esterase enzyme stack. Detection via amperometric H₂O₂ + spectrophotometric.

**BOM.** ~$120 (reader; cartridge $5/test).

**THEORY.** Cholesterol esterase + cholesterol oxidase → H₂O₂ → electrochemical detection. Triglycerides: lipase → glycerol → glycerol oxidase → H₂O₂. HDL/LDL by selective precipitation.

**ASSEMBLY.** Same as CHEM-01 with lipid-specific cartridge.
