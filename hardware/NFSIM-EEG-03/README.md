### NFSIM-EEG-03 — Around-Ear EEG

**README.** cEEGrid-style flexible electrode array around the ear. Discreet, daily-wearable EEG for sleep monitoring, drowsiness alert, BCI in everyday settings.

**SPECIFICATIONS.** 10-channel Ag/AgCl flex array around ear; ADS1299; BLE; 200 mAh battery. ~$95.

**THEORY.** Same EEG principle; flex electrode array conforms to scalp around the ear. Lower amplitude than scalp EEG, but consistent placement (per individual) enables longitudinal monitoring.

**ASSEMBLY.** Custom flex PCB with electrode pattern; electronics housing behind ear.
