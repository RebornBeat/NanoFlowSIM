#### README.md

**Purpose.** Plug-in compute module that turns NFSIM-CON-04 from a relay into a complete standalone AI workstation. Three SoM tiers — Pi CM5 entry-level (~$95), RK3588 mid-range with NPU (~$180), Jetson Orin Nano high-end with CUDA (~$500). Ubuntu 24.04 preinstalled with nfsim-edge-server, ONNX Runtime, Dorado/Bonito basecallers, BART MRI reconstruction, Kilosort, etc.
