### NFSIM-ENV-04 — Air Quality Biomonitor

**README.** Multi-sensor environmental monitor: CO₂ + particulate matter + VOC + NO₂ + humidity + temp + pollen detection (optical particle classifier).

**SPECIFICATIONS.** SCD41 (CO₂) + SPS30 (PM1/2.5/10) + SGP41 (VOC/NOx) + SEN54 (multi-pillar) + SHT40 (T+H). Optional optical pollen classifier (Pi + camera). BLE 5.0. ~$75.

**THEORY.** Indoor air quality affects health (respiratory, allergic, cognitive). Multi-sensor pattern detection identifies issues (combustion, off-gassing, biological contamination).

**ASSEMBLY.** Standard sensor breakout integration in ventilated enclosure.
