### NFSIM-IMPLANT-01 — Subcutaneous Sensor Node

**README.** NFC-powered, batteryless subcutaneous implant (titanium capsule) with multiple sensors (glucose, temperature, motion). External wand provides power + data link.

**SPECIFICATIONS.** Grade 5 titanium hermetic capsule (~3×15 mm); inductively-powered nRF52840 (NFC mode); 13.56 MHz; ISO 15693 protocol. Sensors: lactate or glucose electrochemistry + temperature. Designed for ~10-year implant life. ~$280.

**THEORY.** Batteryless implants powered by external reader (analogous to RFID/contactless payment). Sensor data transmitted on each interrogation. Replaces blood draws for selected chronic monitoring scenarios. **Research-only in MVP** — clinical implantation pathway involves full regulatory approval.

**ASSEMBLY.** Bioglass-sealed Grade 5 Ti capsule; coil + electronics encapsulated; sensor exposed via biocompatible membrane.
