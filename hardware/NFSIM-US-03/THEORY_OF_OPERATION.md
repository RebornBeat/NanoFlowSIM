#### THEORY_OF_OPERATION.md

**PVDF transduction physics.** PVDF (polyvinylidene fluoride) is a piezoelectric polymer. The β-phase crystalline structure has dipoles oriented across the film thickness. Applied voltage causes thickness change → ultrasound emission. Conversely, incident pressure changes thickness → voltage. Coefficient d33 ~20–30 pC/N (vs. PZT ~250 pC/N) but: (a) much wider bandwidth, (b) closer acoustic impedance match to tissue/water (3.5 MRayl vs 30 MRayl for PZT, water = 1.5 MRayl), (c) flexible — wraps cylinder.

**Capsule imaging geometry.** 32 PVDF elements wrap the circumference. Each element fires (transmit) and receives. Synthetic aperture reconstruction on the external belt builds a 360° radial image of the GI wall in slices as the capsule travels. IMU data tracks capsule rotation/translation; the reconstruction de-rotates and assembles a continuous mucosa map.

**Signal chain.**
```
Capsule:
PVDF element (32 channels, multiplexed in groups of 8)
  → low-V pulser (±30V, MAX14808-class)
  → switch to RX
  → 4-channel AFE (TI AD9670 or custom discrete)
  → 12-bit ADC
  → FPGA framing (Lattice iCE40UP5K — ultra-low-power)
  → ESP32-S3 (orchestrator) + storage to micro-SD
  → 434 MHz FSK uplink (chunked at ~1 Mbps)

External belt receiver:
434 MHz Si4463 ×4 (diversity)
  → uart to STM32H7 on belt
  → assembly + forward error correction
  → BLE to phone OR Wi-Fi to hub/laptop/CON-06
  → host runs full synthetic aperture reconstruction + AI denoising
  → image saved + clinical analysis
```

**Provenance.** Raw RF I/Q per element per frame is recorded on the capsule micro-SD (downloaded post-excretion via UART if user retains the capsule, or streamed live via 434 MHz with selectable bandwidth). Reconstruction chain logs synthetic-aperture algorithm version + AI denoiser model + weights hash. **Per-pixel confidence map** is generated by Bayesian neural network denoiser.

#### Firmware Architecture

**ESP32-S3 (orchestrator):**
- **AcquisitionTask** (highest priority): triggers FPGA capture sequences, orchestrates 4 element groups round-robin
- **IMUTask** (high priority, 100 Hz): reads LSM6DSO + LIS3MDL, time-stamps for off-board reconstruction
- **StorageTask**: writes RF I/Q chunks to micro-SD as compressed frames
- **RadioTask**: chunks RF data for 434 MHz uplink with selectable bandwidth modes (low-bw thumbnail mode vs. full-bw streaming)
- **PowerTask**: monitors battery; switches to thumbnail-only mode if battery < 30%; shuts down at < 10%

**FPGA:** raw I/Q deserialization, lossless compression, framing.

**External belt receiver firmware (STM32H7):** Si4463 ×4 diversity receivers + FEC + UART/BLE to phone/hub/CON-06.

**Host (phone/hub/CON-06):**
- Reconstruction service (Rust + GPU)
- AI denoiser (Bayesian U-Net)
- Synthetic aperture image build
- IMU-based motion compensation
- Per-pixel confidence overlay
- Capsule trajectory visualization (3D body map integration)
