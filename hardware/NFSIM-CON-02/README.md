### NFSIM-CON-02 — Universal Cartridge Interface (open spec)

**README.** Open hardware specification for a universal cartridge interface used across NFSIM chemistry/molecular devices. 85×54×2 mm cartridge with 32-pin edge connector + optical windows + barcode/QR code identifier.

**SPECIFICATIONS.** Open spec; no BOM (it's a standard). Compatible devices implement the same connector + protocol.

**THEORY.** Standardized cartridge interface enables ecosystem: any device implementing CON-02 reads any compatible cartridge. Reagents, fluidic channels, electrodes, optical windows on the cartridge; reader handles electronics + optics + heating.

**INCLUDES (in the spec):** 32-pin pinout, optical window positions, barcode standard, RFID option for advanced cartridges.

**ASSEMBLY.** This is the spec, not a device. Documentation, KiCad symbols, mechanical drawings published openly.
